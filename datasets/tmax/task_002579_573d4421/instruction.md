You are an AI assistant helping a data science researcher validate a probabilistic model. The researcher has split their experimental data across three different CSV files in the `/home/user/data/` directory:

1. `/home/user/data/priors.csv` - Contains the `id`, a feature `x`, and the `prior` probability for a hypothesis.
2. `/home/user/data/likelihoods.csv` - Contains the `id`, a feature `y`, and the `likelihood` of the observed data given the hypothesis.
3. `/home/user/data/model_preds.csv` - Contains the `id` and the `predicted_posterior` generated by a black-box model.

Your task is to build a reproducible data processing pipeline (using any language of your choice) to do the following:
1. **Multi-source joining:** Join all three datasets on the `id` column.
2. **Bayesian inference:** Calculate the exact *normalized posterior* probability for each ID. The unnormalized posterior is `prior * likelihood`. The normalized posterior is the unnormalized posterior divided by the sum of all unnormalized posteriors across *all* rows in the joined dataset.
3. **Model output validation:** Identify "flagged" IDs where the absolute difference between your calculated normalized posterior and the model's `predicted_posterior` is strictly greater than `0.01`.
4. **Correlation analysis:** Filter out the flagged IDs. For the remaining *valid* IDs, calculate the Pearson correlation coefficient between feature `x` and feature `y`.

Create a script that performs this pipeline and outputs a final report to `/home/user/report.json` in the following exact format:
```json
{
  "flagged_ids": [list of integers, sorted ascending],
  "correlation": float (rounded to 4 decimal places)
}
```

Ensure your code is self-contained and reproducible. Run your script to generate the `/home/user/report.json` file.
