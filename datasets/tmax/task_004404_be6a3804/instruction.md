You are an incident response analyst investigating a recent server anomaly. You have been provided with two artifacts:
1. A video recording of the server's monitoring dashboard during the incident, located at `/app/server_feed.mp4`.
2. A requirement to process raw JSON-lines logs generated by the affected system.

Your task is to write a robust Bash script at `/home/user/process_logs.sh` that processes a stream of JSON logs from standard input and outputs formatted, anonymized text to standard output. 

Step 1: Video Analysis (Information Extraction)
The video `/app/server_feed.mp4` shows a terminal dashboard. At some point in the video, a prominent 8-character incident code (format: `INC-XXXX`) flashes on the screen. 
Use `ffmpeg` and `tesseract` (both pre-installed) to extract frames and run OCR to find this incident code. You will need this code as a template prefix for the next step.

Step 2: Log Processing Script (Bash, Masking, Templates)
Write a Bash script at `/home/user/process_logs.sh`. Ensure it is executable (`chmod +x`).
The script must read JSON-lines from standard input (`stdin`), one object per line.
Each JSON object has the following schema:
`{"log_id": "string", "ip": "string", "message": "string", "level": "string"}`

For each line, your script must:
1. **Validation Checkpoint:** Ignore any log line where the `level` is not exactly `"CRITICAL"`.
2. **Information Extraction:** Extract the `log_id`, `ip`, and `message` fields. 
3. **Data Masking:** Anonymize the `ip` address by replacing the first two octets with `XXX.XXX` (e.g., `192.168.1.55` becomes `XXX.XXX.1.55`).
4. **Unicode Handling:** Ensure that any Unicode escape sequences (e.g., `\u0041` or `\u2605`) in the `message` field are properly decoded into their literal characters. (Hint: `jq -r` handles this natively).
5. **Template-based Generation:** Print the processed log to standard output (`stdout`) in the exact following format:
`[<INCIDENT_CODE>] ID:<log_id> | IP:<anonymized_ip> | MSG:<decoded_message>`

Replace `<INCIDENT_CODE>` with the exact 8-character code you extracted from the video.

Requirements & Constraints:
- Use standard CLI tools (bash, jq, sed, awk, etc.).
- The script must read from stdin and write to stdout.
- It must perfectly handle valid JSON-lines logs, silently dropping non-CRITICAL logs.
- Do NOT print anything else to standard output in your final script.
