You are an AI assistant helping a scientific researcher process measurements from a novel optical sensor. 

The sensor's system operator matrix $M$ is known to be highly ill-conditioned, meaning that direct inversion to solve for the original signal $x$ from the measurement vector $b$ ($Mx = b$) fails entirely due to numerical instability and noise amplification.

The researcher knows a physical constraint about the true signal: its time-domain representation (obtained via the Inverse Fast Fourier Transform of $x$) has a finite support. Specifically, the second half of the time-domain signal should contain virtually zero energy.

Your task is to implement a robust reconstruction pipeline using Tikhonov regularization and spectral analysis.

**Data provided:**
* `/home/user/M.csv`: The 100x100 system operator matrix $M$ (comma-separated).
* `/home/user/b.csv`: The 100x1 measurement vector $b$ (comma-separated).

*(Note: If the files do not exist, wait for the simulation to finish, or ensure you are in the correct directory. They are pre-generated by the lab equipment).*

**Objective:**
1. You must find the optimal regularization parameter $\lambda$ from the following candidate list: `[0.1, 0.01, 0.001, 0.0001, 0.00001, 0.000001]`.
2. For each $\lambda$, solve the regularized linear system: $(M^T M + \lambda I) x = M^T b$.
3. Compute the time-domain signal $v = \text{IFFT}(x)$ using a standard 1D Inverse Discrete Fourier Transform.
4. Calculate the "tail energy" $E$ of the signal, defined as the sum of the squared absolute values of the second half of $v$ (indices 50 through 99 inclusive): $E = \sum_{i=50}^{99} |v_i|^2$.
5. Select the **smallest** $\lambda$ from the candidate list that results in a tail energy $E < 0.05$.
6. For this selected optimal $\lambda$, find the maximum absolute peak in the first half of the time-domain signal (indices 0 through 49 inclusive): $P = \max_{i \in [0, 49]} |v_i|$.

**Output requirements:**
Create a JSON file at `/home/user/solution.json` containing exactly two keys:
* `"optimal_lambda"`: The float value of the chosen $\lambda$.
* `"max_peak"`: The float value of $P$, rounded to exactly 4 decimal places.

You will need to write a Python script to perform this analysis. You may use standard scientific libraries like `numpy` and `scipy` (install them using `pip` if they are not present in your environment).
