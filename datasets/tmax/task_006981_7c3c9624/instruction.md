You have a local Git repository at `/home/user/repo`. There is a C++ application inside it (`main.cpp`) that connects to a local service to retrieve a configuration string and writes it to a log file. Currently, the deployment fails due to environment differences, missing hooks, and network configuration issues.

Perform the following tasks:

1. **SSH Port Forwarding**: 
   A legacy configuration server is running locally on port `9090`. The C++ application is hardcoded to connect to `localhost:8080`.
   Set up a local SSH tunnel to forward port `8080` to `localhost:9090`. You must run this in the background. (Assume `sshd` is running on port 22. You may need to generate an SSH key and add it to `/home/user/.ssh/authorized_keys` to authenticate as `user@localhost` without a password).

2. **Fix the C++ Application**:
   The source code at `/home/user/repo/main.cpp` attempts to write its output to `output.log` in the current working directory. 
   Modify `main.cpp` so that it always writes to the absolute path `/home/user/deployment_output.log`, regardless of where it is executed from. 
   Also, ensure it properly handles connection errors by returning a non-zero exit code if it cannot connect to `localhost:8080`.

3. **Git Hook Configuration**:
   Create a `post-commit` hook in `/home/user/repo/.git/hooks/post-commit`. 
   This must be a robust Bash script that:
   - Compiles `main.cpp` using `g++` into an executable named `/home/user/repo/app_bin`.
   - Executes `/home/user/repo/app_bin`.
   - If either compilation or execution fails, the script should echo an error and exit with a non-zero status.

4. **Trigger the Pipeline**:
   Commit your changes to `main.cpp` in the `/home/user/repo` repository with the message "Fix paths and harden deployment". This commit should trigger the `post-commit` hook, compile the code, fetch the data via the SSH tunnel, and write to `/home/user/deployment_output.log`.

Do not run the compiled binary manually to generate the output file; it must be generated by the `post-commit` hook triggering upon your commit.
