   Compute the first (largest) singular value of the simulated data ($\sigma_{sim, 1}$) and the first singular value of the baseline data ($\sigma_{base, 1}$). Calculate the ratio: $R = \sigma_{sim, 1} / \sigma_{base, 1}$.

3. **Spectral Analysis:** 
   Extract the first left singular vector ($u_1$) from the SVD of the simulated data (this represents the dominant temporal pattern). Perform a discrete Fourier transform (FFT) on $u_1$. Find the index of the dominant frequency (the index of the maximum magnitude in the power spectrum, ignoring the DC component/zero-frequency).

**Output Format:**
The final output must be generated by your pipeline and saved to `/home/user/analysis_results.json` exactly as follows:
```json
{

}
```
