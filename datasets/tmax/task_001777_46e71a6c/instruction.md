You are a Machine Learning Engineer preparing training data for a new base-calling model. You have been given a synthetic video `/app/base_calls.mp4` generated by an experimental fluorescence sequencer, and a reference dataset `/app/dataset.fasta` containing 10,000 known DNA sequences.

Your task is to extract the nucleotide sequence from the video and find its closest match in the reference dataset.

**Video Specification:**
- The video `/app/base_calls.mp4` contains exactly 50 frames (at 10 FPS).
- Each frame represents a single nucleotide base call and has a resolution of 100x100 pixels.
- The frame is divided into four 50x50 quadrants:
  - Top-Left (x: 0-49, y: 0-49): Adenine (A)
  - Top-Right (x: 50-99, y: 0-49): Cytosine (C)
  - Bottom-Left (x: 0-49, y: 50-99): Guanine (G)
  - Bottom-Right (x: 50-99, y: 50-99): Thymine (T)
- To determine the base for a given frame, you must find the quadrant with the highest average pixel intensity (considering all color channels or converting to grayscale).

**Matching Specification:**
- Once you have reconstructed the 50-bp query sequence from the video, you must align it against the sequences in `/app/dataset.fasta`.
- Use the Hamming distance to find the sequence in the FASTA file that is most similar to your query sequence.
- **Implementation Constraint:** You must implement the FASTA parsing, sequence comparison, and the search logic in **Rust**. Your Rust program must utilize parallel computing (e.g., the `rayon` crate) to compare the query against the dataset concurrently. 

**Output Requirement:**
Your Rust program should write the results to `/home/user/result.json` in the following exact format:
```json
{
  "query": "ACGTACGT...",
  "best_match_header": "SEQ_01234",
  "hamming_distance": 2
}
```

Write the necessary Rust code, run it, and generate the `result.json` file. You may use shell tools (like `ffmpeg`) to extract the frames from the video before processing them with your Rust program or Python scripts, but the core sequence alignment and parallel FASTA search must be in Rust.
