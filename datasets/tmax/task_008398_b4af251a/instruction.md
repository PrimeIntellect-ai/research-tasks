You are a data analyst tasked with processing a massive dataset of sensor metrics generated by an ETL pipeline. Unfortunately, the pipeline frequently retried failed batches, resulting in duplicate timestamps in the exported CSV files. 

Your task is to write a highly efficient C program that streams the data, deduplicates it on the fly, buckets the metrics into time intervals, and computes a rolling mathematical aggregation. 

**Data Details:**
- Input file: `/home/user/data/metrics.csv`
- The file has a header row: `timestamp,value`
- `timestamp` is a 64-bit integer (Unix epoch).
- `value` is a double-precision floating-point number.
- The rows are globally sorted by `timestamp` in ascending order.
- **Deduplication:** Due to ETL retries, there are consecutive rows with the exact same `timestamp`. You must only process the *first* occurrence of any `timestamp` and strictly ignore any subsequent rows with that same `timestamp`.

**Processing Requirements:**
You must write a C program located at `/home/user/app/processor.c` that does the following:
1. **Stream Processing:** Read the input file line-by-line. Do not load the entire file into memory (assume the file could be larger than available RAM).
2. **Time-Based Bucketing:** Group the deduplicated records into 10-second buckets. A bucket's identifier (and start time) is `timestamp - (timestamp % 10)`. For each bucket, calculate the arithmetic mean (average) of the `value`s within it.
3. **Rolling Aggregation:** For each bucket, calculate a 3-period moving average of the bucket averages. This means the moving average for the current bucket is the average of the current bucket's mean and the up to 2 strictly previous *non-empty* buckets' means. (If it's the first bucket, the moving average equals the bucket average. If it's the second, it's the average of the first and second).

**Output format:**
Your program should output a CSV file at `/home/user/data/output.csv` with the following header:
`bucket_ts,bucket_avg,moving_avg`

Each subsequent line should represent a bucket (in chronological order) and format the floats to exactly 4 decimal places. 
Example line:
`1700000010,45.1234,42.5678`

**Constraints & Execution:**
- Write only standard C (C99 or C11).
- Compile your program with `gcc -O3 /home/user/app/processor.c -o /home/user/app/processor`.
- Execute your program to generate `/home/user/data/output.csv`.
