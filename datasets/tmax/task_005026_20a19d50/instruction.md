     `Loaded <kind>: <name>`

   ```yaml

   kind: Deployment
   metadata:
     name: frontend-app
   spec:
     replicas: 3
   ```
   Add, commit, and push this file to the bare repository to trigger the pipeline end-to-end. 

Ensure that by the end of your workflow, `/home/user/cluster-status.log` exists and contains the correct output log generated by your operator responding to the webhook.
