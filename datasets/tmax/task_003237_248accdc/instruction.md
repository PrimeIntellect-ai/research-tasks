5. **Final Pipeline:**
   Create a bash script at `/home/user/run_pipeline.sh` that:
   - Runs `/home/user/fetch.exp`.
   - Pipes the output to `/home/user/aggregator`.
   - Saves the final JSON output to `/home/user/metrics_out.json`.

Ensure your C++ program compiles without errors and your pipeline script executes successfully, creating `/home/user/metrics_out.json` with the correct JSON.
