   - It must decode the Base64 string into raw bytes.
   - It must load `/home/user/libchecksum.so` via `ctypes` and pass the raw bytes to `compute_weighted_checksum`.
   - It must return a JSON response: `{"checksum": <integer_result>}`.

3. **Property-Based Testing (Python)**
   Write a test script at `/home/user/test_api.py`.

   - Base64-encode the bytes.
   - Send a POST request to `http://127.0.0.1:8080/checksum`.
   - Assert that the API responds with a 200 OK and the returned checksum matches the expected checksum.

4. **Execution and Logging**
   Start your Flask API in the background.
   Run your tests using `pytest /home/user/test_api.py -v` and redirect the output to `/home/user/test_results.log`.
