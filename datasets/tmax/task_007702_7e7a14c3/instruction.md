You are acting as a Machine Learning Engineer preparing a dataset for a Physics-Informed Neural Network (PINN). You have raw accelerometer data from a simulated moving object, but your neural network requires velocity and position data as well. 

You need to integrate the acceleration data to compute velocity and position. However, since numerical integration can accumulate errors (numerical instability), you must test two different integration methods, compare them against a small reference dataset, and use the most accurate method to generate your final training data.

Here are your instructions:

1. **Input Data**: 
   - `/home/user/raw_sensor.csv`: Contains two columns, `t` (time) and `a` (acceleration). Time ranges from 0 to 10 seconds.
   - `/home/user/reference_trajectory.csv`: Contains a high-fidelity reference for the first 2 seconds. Columns are `t`, `v_ref` (velocity), and `p_ref` (position).

2. **Integration Implementation**:
   Write a Python script (e.g., `/home/user/process_data.py`) that reads `raw_sensor.csv` and calculates velocity `v` and position `p` using two methods:
   - **Method 1: Forward Euler Method**
     - $v_{i+1} = v_i + a_i \cdot \Delta t$
     - $p_{i+1} = p_i + v_i \cdot \Delta t$
   - **Method 2: Trapezoidal Rule**
     - $v_{i+1} = v_i + \frac{a_i + a_{i+1}}{2} \cdot \Delta t$
     - $p_{i+1} = p_i + \frac{v_i + v_{i+1}}{2} \cdot \Delta t$
   - *Assume initial conditions:* $v_0 = 0.0$ and $p_0 = 0.0$ at $t = 0$. $\Delta t$ can be derived from the time column (it is uniform).

3. **Reference Comparison & Stability Testing**:
   - Compare the results of both methods against the reference dataset (`reference_trajectory.csv`) for the matching time steps (t=0 to t=2.0).
   - Calculate the Mean Squared Error (MSE) for velocity and position for both methods.
   - Create a JSON file at `/home/user/integration_report.json` with the exact following keys:
     - `"euler_mse_v"`: MSE of Euler velocity vs reference.
     - `"euler_mse_p"`: MSE of Euler position vs reference.
     - `"trapz_mse_v"`: MSE of Trapezoidal velocity vs reference.
     - `"trapz_mse_p"`: MSE of Trapezoidal position vs reference.
     - `"best_method"`: A string, either `"euler"` or `"trapz"`, depending on which method has the lower combined MSE (MSE_v + MSE_p).

4. **Final Dataset Generation**:
   - Using the data generated by the `"best_method"` over the *entire* time range (t=0 to 10), output the final dataset to `/home/user/training_data.csv`.
   - The CSV must have exactly these columns in order: `t,a,v,p`.
   - Round all values in the final CSV to 4 decimal places.

Ensure your Python script runs without errors and produces the two required output files (`integration_report.json` and `training_data.csv`). You can install any required Python packages (like pandas or numpy) using pip.
