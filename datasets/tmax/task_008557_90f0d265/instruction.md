You are acting as a computational biology software engineer. A researcher needs a Rust application to simulate a multiplex amplification reaction using a custom Ordinary Differential Equation (ODE) model, perform convergence testing on the solver, and compare the resulting amplicon distribution to experimental data.

Please create a Rust project in `/home/user/pcr_sim` to solve this task.

**Phase 1: Primer Sequence Alignment**
You are given a template sequence in `/home/user/template.txt` and a list of forward primers in `/home/user/primers.txt` (one per line). 
1. Find the first exact substring match (0-indexed) for each primer in the template.
2. The length of the amplicon generated by a primer is defined as `Template_Length - Match_Index`. (Assume primers only bind once, and only on the forward strand).

**Phase 2: ODE Numerical Solving**
Model the amplification of these amplicons using the following system of ODEs:
For each amplicon $i$:
$$ \frac{dA_i}{dt} = k \cdot A_i \cdot \left(1 - \frac{\sum_{j} A_j}{C_{max}}\right) $$
Where:
* $A_i(t)$ is the concentration of amplicon $i$.
* Initial conditions: $A_i(0) = 1.0$ for all $i$.
* $k = 0.8$ (amplification rate).
* $C_{max} = 100,000.0$ (carrying capacity).
* Simulate from $t = 0$ to $t = 30$.

Implement the Forward Euler method to solve this system.

**Phase 3: Convergence Testing**
The researcher needs to balance speed and accuracy. Test the following time step sizes ($\Delta t$): `[1.0, 0.5, 0.1, 0.05, 0.01]`.
First, compute a "baseline" total concentration at $t=30$ using $\Delta t = 0.001$.
Find the largest $\Delta t$ from the test list that results in a total final concentration ($\sum A_i(30)$) that is within `0.5%` of the baseline's total final concentration (i.e., $\frac{|Total_{\Delta t} - Total_{baseline}|}{Total_{baseline}} \le 0.005$).

**Phase 4: Probability Distribution Distance**
Using the final concentrations $A_i(30)$ from your *chosen* optimal $\Delta t$:
1. Normalize the concentrations so they sum to 1.0 to form a probability distribution $P$.
2. Read the experimental distribution $Q$ from `/home/user/experimental_dist.json` (a map of amplicon length to probability).
3. Compute the Total Variation (TV) distance between $P$ and $Q$. The TV distance is defined as: $TV(P, Q) = \frac{1}{2} \sum_{x} |P(x) - Q(x)|$, where $x$ iterates over all amplicon lengths present in either distribution. (Treat missing lengths as having probability 0.0).

**Output:**
Write a JSON file to `/home/user/simulation_results.json` with the following structure:
```json
{
  "optimal_dt": 0.1,
  "amplicon_lengths": [800, 500, 200],
  "final_total_concentration": 99543.21,
  "tv_distance": 0.045
}
```
*Note: The values above are just examples.* Ensure your final concentrations and TV distance are rounded to 4 decimal places.
