You are a data analyst working on a legacy server where the standard data science tools (Python, R) have been completely removed due to a security audit. You need to perform statistical analysis using only standard GNU Linux command-line utilities (Bash, awk, sed, shuf, bc, etc.). 

A previous analyst left a broken pipeline that produced blank outputs. Your task is to build a robust Bash script from scratch that calculates the Pearson correlation coefficient between two variables using bootstrap sampling to estimate the mean and standard deviation of the correlation.

**Dataset:**
You have a dataset at `/home/user/dataset.csv`. 
It has three columns: `id`, `feature_A`, and `feature_B`. The first row is the header.

**Your Task:**
Write a Bash script at `/home/user/bootstrap_corr.sh` that takes exactly one integer argument: the number of bootstrap iterations `N`.

The script must perform the following algorithm:
1. Read `/home/user/dataset.csv` (ignore the header).
2. For `N` iterations, create a bootstrap sample. A bootstrap sample is created by randomly sampling rows from the original dataset **with replacement**. The size of the sample must equal the number of data rows in the original dataset.
3. For each bootstrap sample, calculate the Pearson correlation coefficient ($r$) between `feature_A` and `feature_B`.
   The formula for Pearson correlation is:
   $r = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum (x_i - \bar{x})^2 \sum (y_i - \bar{y})^2}}$
4. Collect the $N$ correlation coefficients.
5. Calculate the mean and the population standard deviation of these $N$ correlation coefficients.
6. Write the final mean and standard deviation to `/home/user/result.txt` in the following exact format (rounded to 4 decimal places):
   ```
   Mean: 0.XXXX
   SD: 0.XXXX
   ```

**Constraints:**
- You must use Bash and standard utilities (like `awk`, `shuf`, `sort`, `bc`). 
- Python, R, Perl, and other scripting languages are disabled on this system.
- Ensure numerical accuracy. Be careful with integer division. Use `awk` or `bc -l` for floating-point math.
- The script must be executable (`chmod +x /home/user/bootstrap_corr.sh`).
- You should run your script with 500 iterations (`./bootstrap_corr.sh 500`) to generate the final `/home/user/result.txt` before finishing the task.

**Deliverables:**
1. The executable script `/home/user/bootstrap_corr.sh`.
2. The output file `/home/user/result.txt` generated by running `./bootstrap_corr.sh 500`.
