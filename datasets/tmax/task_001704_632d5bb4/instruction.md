You are acting as a penetration tester investigating a compromised Linux server. 

An attacker exploited a file upload vulnerability (susceptible to path traversal and improper validation) to upload a malicious executable to the web application's upload directory. The incident data has been copied to your local machine at `/home/user/incident`.

Your objectives are to analyze the uploaded files, reverse-engineer the backdoor's authentication mechanism, and audit the firewall configuration left by the attacker.

Perform the following steps using Bash and standard Linux utilities:

1. **Binary Analysis**: Analyze the files in `/home/user/incident/uploads`. One of the files is secretly an ELF binary disguised with an image extension. Identify this malicious file.
2. **Credential Extraction**: The disguised ELF binary contains hardcoded strings for a backdoor login mechanism. Extract the SHA-256 hash and the salt from the binary. They are stored as strings in the format `BDOOR_HASH=<sha256_hex>` and `BDOOR_SALT=<salt_string>`.
3. **Password Cracking**: The attacker's password is a 4-digit numeric PIN (from `0000` to `9999`). The hash was generated by concatenating the PIN and the salt (e.g., if PIN is `1234` and salt is `abcd`, the hashed content is `1234abcd` without a newline). Write and execute a Bash script to brute-force the 4-digit PIN that matches the extracted SHA-256 hash.
4. **Service Auditing**: The attacker modified the firewall to allow backdoor access. Audit the script `/home/user/incident/firewall.sh` to determine which TCP port was explicitly opened for the attacker's IP address: `10.10.5.55`.

Once you have gathered all the information, create a log file at `/home/user/resolution.log` with the exact following format:

```
MALICIOUS_FILE: <filename_with_extension>
PIN: <cracked_4_digit_pin>
PORT: <port_number>
```

Replace `<filename_with_extension>` with just the name of the malicious file (e.g., `avatar.png`), `<cracked_4_digit_pin>` with the 4-digit PIN you cracked, and `<port_number>` with the extracted port.
