You are a data scientist building a highly performant, reproducible pipeline in C to test statistical hypotheses on large datasets of sensor readings.

You have been provided with two binary data files in the directory `/home/user/data/`:
- `dataset_1.bin`
- `dataset_2.bin`

Each file contains exactly 10,000 double-precision floating-point numbers (IEEE 754, 64-bit, little-endian).

Your task is to write a C program that reads a dataset and determines whether it was more likely generated by a Normal distribution or a Laplace distribution, using Maximum Likelihood Estimation (MLE). 

Perform the following steps:

1. Create a working directory at `/home/user/workspace/`. All your source code must go here.
2. Write a C program named `fit_models.c`. The program must take a single command-line argument: the path to a binary `.bin` dataset.
3. The program must read the 10,000 `double` values and compute the maximum likelihood parameters for two models:
   - **Model 1: Normal Distribution**
     - $\mu = \text{mean}(X)$
     - $\sigma^2 = \frac{1}{N} \sum (x_i - \mu)^2$
     - Log-likelihood: $LL_{norm} = -\frac{N}{2}\ln(2\pi\sigma^2) - \frac{N}{2}$
   - **Model 2: Laplace Distribution**
     - $\mu_{lap} = \text{median}(X)$ *(Note: For N=10000, use the average of the elements at sorted indices 4999 and 5000)*
     - $b = \frac{1}{N} \sum |x_i - \mu_{lap}|$
     - Log-likelihood: $LL_{lap} = -N\ln(2b) - N$
4. The program should compare $LL_{norm}$ and $LL_{lap}$. It must print a single line to standard output in this exact comma-separated format:
   `[dataset_name],[Best_Model],[Log_Likelihood_Difference]`
   - `[dataset_name]` is the filename without the directory path or `.bin` extension (e.g., `dataset_1`).
   - `[Best_Model]` is either `Normal` or `Laplace`.
   - `[Log_Likelihood_Difference]` is the absolute difference $|LL_{norm} - LL_{lap}|$, formatted to exactly 4 decimal places.
5. Write a `Makefile` in `/home/user/workspace/` that compiles `fit_models.c` into an executable named `fit_models`. Ensure you link the math library.
6. Write a bash script `/home/user/workspace/run_pipeline.sh` that:
   - Compiles the code using your `Makefile`.
   - Runs the executable on `dataset_1.bin` and `dataset_2.bin` (in that order).
   - Redirects and saves the combined standard output into a final report file at `/home/user/results.csv`.

Ensure your C code is robust and your pipeline can be executed simply by running `bash /home/user/workspace/run_pipeline.sh`.
