You are a bioinformatics analyst developing a new structural analysis pipeline. We have a dictated audio file from a researcher containing an amino acid sequence. You need to build a system that processes this sequence, performs a parallelized Monte Carlo simulation in C++ to assess its folding energy significance, and serves the results via a web API.

Step 1: Transcription
An audio file is located at `/app/experiment_dictation.wav`. It contains a spoken sequence of amino acid names (e.g., "Alanine Cysteine..."). Determine the corresponding 1-letter amino acid sequence (e.g., "AC...").

Step 2: C++ Simulation Engine
Write a C++ program using OpenMP that calculates a custom "Folding Energy" and estimates its p-value via Monte Carlo permutation testing. 
The properties for the relevant amino acids (Mass, Hydrophobicity) are:
M: (149, 1.9)
A: (89, 1.8)
S: (105, -0.8)
G: (75, -0.4)
Y: (181, -1.3)

For any given sequence of length N, the Folding Energy $E$ is defined as:
$E = \sum_{i=1}^{N-1} \left( |Mass_{i} - Mass_{i+1}| \times (Hydro_{i} + Hydro_{i+1}) \right)$

Your C++ program must:
1. Accept the 1-letter sequence string as a command-line argument.
2. Calculate the original sequence's energy $E_{orig}$.
3. Run a Monte Carlo simulation generating exactly 1,000,000 random permutations of the given sequence. Use OpenMP to parallelize this loop.
4. For each permutation, calculate its energy $E_{perm}$.
5. Calculate the p-value as the fraction of permutations where $E_{perm} \le E_{orig}$.
6. Output the results to stdout as JSON: `{"original_energy": <float>, "p_value": <float>}`.

Step 3: API Service
Create an HTTP server (you may use Python, e.g., Flask/FastAPI, or a C++ framework) that listens on `127.0.0.1:8000`.
It must expose a `POST` endpoint at `/analyze`.
- Request: JSON object `{"sequence": "MASGY"}`
- Action: The server invokes your compiled C++ binary with the sequence.
- Response: Returns the JSON generated by the C++ binary.

Start the server in the background and ensure it is ready to accept requests. Keep the server running.
