You are an artifact manager tasked with curating a binary repository and preparing a unified metadata backup. The repository is located at `/home/user/artifact_repo`. 

A previous attempt to index and back up this repository failed because the repository structure contains circular symbolic links (symlink loops) that caused the backup script to crash with infinite recursion. Additionally, the metadata files accompanying the binaries were generated by different legacy systems, meaning they are in various formats (JSON, XML, CSV) and character encodings (UTF-8, UTF-16, ISO-8859-1).

Your task is to write and execute a Python script that traverses the repository, safely bypasses any symlink loops, reads and normalizes the metadata files, and produces a unified incremental backup manifest.

Requirements for the script:
1. **Safe Traversal**: Traverse `/home/user/artifact_repo` recursively. You must detect and skip symlinks that lead to infinite loops or previously visited directories to prevent recursion errors.
2. **Metadata Parsing & Encoding**: 
   - Identify files named exactly `metadata.json`, `metadata.xml`, or `metadata.csv`.
   - Read the files. Be prepared to handle UTF-8, UTF-16, and ISO-8859-1 encodings. You must correctly decode the file regardless of which of these three encodings it uses.
3. **Data Extraction**: Extract the following properties from each metadata file:
   - `artifact_id` (string)
   - `version` (string)
   - `checksum` (string)
   *Note on formats*: 
   - JSON format has these at the root level (e.g., `{"artifact_id": "...", "version": "...", "checksum": "..."}`).
   - XML format has a root `<metadata>` element containing `<artifact_id>`, `<version>`, and `<checksum>`.
   - CSV format has a header row with columns `artifact_id,version,checksum` and a single data row.
4. **Manifest Generation**: Output the aggregated metadata into a single JSON Lines file at `/home/user/backup_manifest.jsonl`. 
   - Every line in this file must be a valid JSON object with four keys: `filepath` (the absolute path to the metadata file), `artifact_id`, `version`, and `checksum`.
   - All text in the output must be strictly UTF-8 encoded.
   - Sort the output lines alphabetically by the `filepath` key so the final file is deterministic.

Before running your script, ensure you have installed any necessary dependencies (e.g., `xmltodict` or similar if you choose to use it, though standard library is sufficient). Once your script generates `/home/user/backup_manifest.jsonl`, the task is complete.
